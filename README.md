# 🛡️ Overwatch 2 Hero Dashboard

A high-performance, responsive web application for exploring Overwatch 2 heroes data. Built with **Next.js**, **Tailwind CSS**, and **JavaScript**, fetching real-time data from the OverFast API.

🔗 **Live Demo:** [overwatch-board.vercel.app](https://overwatch-board.vercel.app/)

---

## ✨ Features

- **Real-time Data:** Fetches the latest hero information including roles and portraits.
- **Pixel Perfect UI:** Meticulously crafted responsive design using Tailwind CSS.
- **Performance Optimized:** Leveraging Next.js Server Components for fast initial page loads.
- **Role Filtering:** Easily browse heroes by their specific roles (Tank, Damage, Support).
- **Gaming Aesthetics:** Modern UI designed with a gaming-inspired theme.

## 🚀 Tech Stack

- **Framework:** [Next.js 14/15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** [JavaScript](https://js.org/index.html)
- **Data Source:** [OverFast API](https://overfast-api.tekrop.fr/)
- **Deployment:** [Vercel](https://vercel.com/)
